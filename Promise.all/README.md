## 요구사항 정리 및 접근방법
여러개의 값(또는 Promise)를 받아 모두 성공하면 결과 배열을 resolve, 아니면 reject하는 함수 구현

접근
먼저 여러개의 값또는 promise 를 입력으로 받고 결과를 추적하는 카운터를 둔다. 결과를 저장할 배열을 입력 길이만큼 생성하고, 입력의 각 요소를 순회하면서 배열은 입력 순서를 유지한 인덱스에 값을 저장하며 카운터를 감소시킨다. 0 이 되면 전체 결과 배열로 resolve한다. 
순회 중 하나라도 실패한다면 reject, 빈 iterable일 경우 빈배열 반환하고 즉시 resolve, 일반 값과 promise 를 동일하게 처리하기위해 각 입력을 Promise 기반으로 통일해서 처리한다.
